## 所有頁面

| 路由 | 功能 | 說明 |
|------|------|------|
| `/` | 首頁 | 平台介紹與入口 |
| `/login` | 登入 | 帳號登入 |
| `/register` | 註冊 | 建立新帳號 |
| `/chat` | AI 對話 | 開始新對話 |
| `/chat/[id]` | 對話詳情 | 查看特定對話 |
| `/dashboard` | 儀表板 | 知識庫統計、快速導航 |
| `/knowledge` | 知識庫 | 文件管理、上傳、搜尋 |
| `/canvas` | Canvas | 富文本編輯器 + AI Copilot |
| `/diagram` | 圖表編輯 | 圖表建立與編輯 |
| `/reports/[id]` | 報告 | 查看報告內容 |
| `/settings` | 設定 | 帳號、Telegram、其他設定 |

## 導航提示

- 左側導航欄可以快速切換各功能模組
- 支援完整的深色模式（Dark Mode）
- 行動裝置自適應介面
