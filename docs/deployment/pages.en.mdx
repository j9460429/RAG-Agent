## All Pages

| Route | Function | Description |
|-------|----------|-------------|
| `/` | Home | Platform introduction and entry point |
| `/login` | Login | Account login |
| `/register` | Register | Create new account |
| `/chat` | AI Chat | Start new conversation |
| `/chat/[id]` | Chat Detail | View specific conversation |
| `/dashboard` | Dashboard | Knowledge base statistics, quick navigation |
| `/knowledge` | Knowledge Base | Document management, upload, search |
| `/canvas` | Canvas | Rich text editor + AI Copilot |
| `/diagram` | Diagram Editor | Diagram creation and editing |
| `/reports/[id]` | Reports | View report content |
| `/settings` | Settings | Account, Telegram, other settings |

## Navigation Tips

- Use the left sidebar for quick navigation between modules
- Full dark mode support
- Mobile-responsive interface
