## 什麼是 NexusMind？

NexusMind 是一個基於 AI 的智慧知識管理平台，核心理念是讓你的知識「活起來」。上傳文件後，AI 不只是儲存，而是理解、關聯、並在你需要時主動提供洞見。

## 核心價值

- **智慧知識庫**：上傳任何格式的文件，AI 自動解析、嵌入向量、建立知識圖譜
- **RAG 增強對話**：對話時自動引用你的知識庫，回答有根據、可溯源
- **🧠 雙層對話記憶系統**：自動從對話中提取使用者的偏好與事實，讓 AI 越用越懂你
- **🎭 設定與角色切換**：豐富的 AI 角色切換與自訂 Persona 管理
- **🎨 Canvas 與圖表**：AI 不只能回答，還能產出文件並使用 Cytoscape 實作互動式流程圖與心智圖
- **🛠️ 隔離沙盒技能系統**：在 Docker 容器內執行程式碼，產出深度報告與 Excel / DOCX 等多格式檔案
- **Telegram 機器人**：隨時隨地透過手機與 AI 互動

## 技術棧

| 類別 | 技術 |
|------|------|
| **前端** | Next.js 16、React 19、TypeScript 5、Tailwind CSS 4 |
| **AI SDK** | Vercel AI SDK（Google、Anthropic、OpenAI） |
| **編輯器** | TipTap 3（ProseMirror） |
| **狀態管理** | Zustand |
| **資料庫** | Supabase（PostgreSQL + pgvector + RLS） |
| **微服務** | Marker、LightRAG、Skill Executor |
| **容器化** | Docker Compose |
| **可觀測性** | Langfuse + OpenTelemetry |
