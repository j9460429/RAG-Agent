## 概述

NexusMind 提供完整的提示詞模板系統，內建 10 種專業 AI 角色預設，支援自訂模板、變數插值、公開分享與收藏功能。

## 內建 AI 角色

系統預設 10 種專業角色，覆蓋常見工作場景：

| 角色 | 專長 | 適用場景 |
|------|------|---------|
| 專業技術顧問 | 架構設計、效能優化 | 技術決策、Code Review |
| 創意文案寫手 | 行銷心理學、平台適配 | 社群貼文、廣告文案 |
| 數據分析專家 | 統計方法、商業洞察 | 資料分析、報告解讀 |
| 程式碼審查員 | SOLID 原則、安全性 | PR Review、程式碼品質 |
| 學習導師 | 費曼技巧、知識鷹架 | 學習新概念、教學 |
| 產品經理助手 | PRD、優先排序、OKR | 產品規劃、需求分析 |
| 簡報大師 | 金字塔原理、故事化表達 | 簡報製作、提案 |
| UI/UX 設計顧問 | 以人為本設計、無障礙性 | 介面設計、體驗優化 |
| SEO 優化專家 | 技術/內容/站外 SEO | 網站優化、內容策略 |
| 敏捷教練 | Scrum/Kanban、團隊成熟度 | 專案管理、流程改進 |

## 使用提示詞模板

### 在對話中使用

1. 在聊天頁面點擊「提示詞」按鈕
2. 瀏覽或搜尋可用模板
3. 點擊模板，系統自動套用該角色的 System Prompt
4. 如模板含變數，填入變數值後開始對話

### 模板變數

模板支援動態變數插值：

```json
{
  "variables": [
    { "name": "topic", "placeholder": "請輸入主題" },
    { "name": "audience", "placeholder": "目標受眾" }
  ]
}
```

使用時系統會提示填入變數值，自動替換到 System Prompt 中。

## 自訂模板

### 建立新模板

| 欄位 | 必填 | 說明 |
|------|------|------|
| `name` | 是 | 模板名稱 |
| `description` | 否 | 描述說明 |
| `icon` | 否 | 圖示（lucide-react icon 名稱，預設 `Sparkles`） |
| `category` | 是 | 分類（預設 `custom`） |
| `system_prompt` | 是 | 系統提示詞內容 |
| `is_public` | 否 | 是否公開分享（預設 `false`） |
| `tags` | 否 | 標籤陣列（用於搜尋與分類） |
| `variables` | 否 | 變數定義陣列 |

### 範例

```json
{
  "name": "技術部落格寫手",
  "description": "專門撰寫技術部落格文章，兼顧深度與可讀性",
  "icon": "PenTool",
  "category": "custom",
  "system_prompt": "你是一位經驗豐富的技術部落格作者。針對 {{topic}} 主題，撰寫面向 {{audience}} 的文章...",
  "is_public": true,
  "tags": ["寫作", "技術", "部落格"],
  "variables": [
    { "name": "topic", "placeholder": "技術主題" },
    { "name": "audience", "placeholder": "初學者/中級/進階" }
  ]
}
```

## 分享與收藏

### 公開分享

將模板設為 `is_public: true` 後，所有使用者皆可看到並使用你的模板。

### 收藏模板

- 只能收藏公開模板
- 收藏的模板會出現在「我的收藏」分類中
- 收藏操作為 toggle：再次點擊取消收藏

### 使用次數統計

每次使用模板時，`usage_count` 自動 +1，用於推薦排序和熱度統計。

## 助理預設

除了提示詞模板，NexusMind 還提供 **AI 助理預設**（Assistant Presets），這是預設的對話角色：

| 預設 | 說明 |
|------|------|
| 一般助手 | 通用型 AI 助手 |
| 寫作助手 | 專注文字創作與編輯 |
| 程式助手 | 專注程式碼開發與除錯 |
| 分析助手 | 專注數據分析與解讀 |
| 創意助手 | 專注創意發想與腦力激盪 |

助理預設與提示詞模板的區別：

| 項目 | 助理預設 | 提示詞模板 |
|------|---------|-----------|
| 數量 | 5 個固定 | 無限自訂 |
| 可編輯 | 否 | 是 |
| 變數 | 無 | 支援 |
| 分享 | 所有使用者皆可用 | 可選擇公開 |
| 用途 | 快速切換 AI 角色 | 精細控制 AI 行為 |

## API 端點

| 操作 | 端點 | 說明 |
|------|------|------|
| 列出模板 | `GET /api/prompts` | 支援 `scope` 參數：`mine`/`public`/`featured`/`chat` |
| 建立模板 | `POST /api/prompts` | 建立自訂模板 |
| 更新模板 | `PUT /api/prompts/:id` | 更新自己的模板 |
| 刪除模板 | `DELETE /api/prompts/:id` | 刪除模板 |
| 使用統計 | `POST /api/prompts/:id/use` | `usage_count += 1` |
| 切換收藏 | `POST /api/prompts/:id/favorite` | Toggle 收藏狀態 |
